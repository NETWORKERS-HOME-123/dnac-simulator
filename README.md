## ✨ Features

- **SD-Access Fabric Provisioning** — define underlay + overlay, onboard switches, configure border + control plane nodes
- **Intent-Based Policy** — group-based access control with policy enforcement
- **Network Assurance** — health dashboards, client experience scoring, root cause analysis
- **Workflow Simulation** — Site hierarchy → Network settings → Device discovery → Fabric design → Policy
- **No Backend Required** — runs entirely in browser, free, no signup

## 🚀 Try It

```bash
git clone https://github.com/NETWORKERS-HOME-123/dnac-simulator.git
cd dnac-simulator
npm install
npm run dev
```

Open `http://localhost:5173` and start practicing.
